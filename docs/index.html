Tokenmaxxing — the 2026 craze of burning AI tokens to look productive. Receipts is the fun way to play along: see how you actually use Claude Code, what you got done, the hours saved — and turn any of it into a card you can share.

Want your own?

Receipts is a free, local tool. It runs on your machine and shows what you get done with Claude Code — no account, nothing uploaded.

  1. 1
    Get Claude Code — Anthropic's coding agent.
    Get Claude Code →
    Already have it? Skip to step 2.
  2. 2
    In Claude Code, add the Receipts marketplace:
    /plugin marketplace add abhitsian/receipts
  3. 3
    Install the plugin:
    /plugin install receipts@receipts
  4. 4
    Open your dashboard any time:
    /receipts:open
